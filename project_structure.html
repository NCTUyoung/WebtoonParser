Electron 架构

Electron 应用由三个主要部分组成：

Main Process (主进程)

负责应用生命周期管理、原生功能、窗口创建等

使用 Node.js 环境

文件位置: src/main/

⟷

Preload Scripts (预加载脚本)

主进程和渲染进程之间的桥梁

提供安全的 API 访问

文件位置: src/preload/

⟷

Renderer Process (渲染进程)

负责 UI 渲染和用户交互

使用 Web 技术 (HTML, CSS, JS)

文件位置: src/renderer/

进程间通信 (IPC) 允许主进程和渲染进程之间安全地交换信息
